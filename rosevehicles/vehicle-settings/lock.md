# Lock

```yaml
Lock:
  Enable: true
  Cooldown: 1000 #mils
  Permission: ""
  Events:
    No_Permission:
      Mechanics: []
    Cooldown:
      Mechanics: []
    Lock:
      Mechanics: []
    Unlock:
      Mechanics: []
```

## Enable

Enable or Disable this option for vehicle

## Cooldown

Delay to prevent spamming by closing/opening vehicle
